# Math Visualization - Toán Lý Số

Ứng dụng visualization các khái niệm toán học trong Huyền học.

## Nguồn gốc
Thư mục gốc: `math_viz/`

## Cấu trúc
```
math-viz/
├── index.html          # Trang chính
├── css/                # Styles
└── js/                 # Scripts
```

## Chạy
```bash
cd math_viz
python -m http.server 8083
```

Truy cập: http://localhost:8083

## Tính năng
- Visualization Ngũ Hành relationships
- Thiên Can, Địa Chi mappings
- Mathematical patterns in Tử Vi
