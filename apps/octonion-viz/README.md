# Octonion Visualization - Bát Nguyên Số

Ứng dụng visualization Octonion / Bát nguyên số.

## Nguồn gốc
Thư mục gốc: `octonion_viz/`

## Cấu trúc
```
octonion-viz/
├── index.html          # Trang chính
├── css/                # Styles
└── js/                 # Scripts
```

## Chạy
```bash
cd octonion_viz
python -m http.server 8084
```

Truy cập: http://localhost:8084

## Tính năng
- Octonion algebra visualization
- 8-dimensional number system
- Bát Quái → Octonion mapping
